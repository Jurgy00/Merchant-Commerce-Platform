# Merchant Commerce Platform;Jurgens Bookshelf

## Tech Stack

- Frontend: Nuxt 4, TypeScript
- Backend:  Spring Boot
- Database: H2 (development), PostgreSQL to be implemented later
- Payments: M-Pesa STK

## How to run frontend

cd frontend
npm install
npm run dev

## How to run backend

cd backend
./mvnw spring-boot:run
